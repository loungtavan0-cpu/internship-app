# 📔 สมุดบันทึกฝึกงาน (Internship Logbook)

เว็บแอปสำหรับบันทึกการฝึกงานประจำวัน สร้างด้วย **HTML + Bootstrap 5 + Vanilla JavaScript**
เปิดใช้งานได้ทันทีใน VS Code โดยไม่ต้อง build หรือติดตั้ง framework ใด ๆ

## 🚀 วิธีเปิดใช้งานใน VS Code

1. เปิดโฟลเดอร์นี้ใน VS Code
2. ติดตั้งส่วนขยาย **Live Server** (โดย Ritwick Dey) จาก Extensions
3. คลิกขวาที่ `index.html` → **Open with Live Server**
4. เบราว์เซอร์จะเปิดที่ `http://127.0.0.1:5500` (หรือพอร์ตใกล้เคียง) — ใช้งานได้ทันที

> ⚠️ อย่าเปิดไฟล์ `index.html` โดยตรงแบบ `file://` เพราะ Service Worker (PWA) และบางฟีเจอร์จะทำงานไม่ได้ ต้องรันผ่าน server (เช่น Live Server) เสมอ

### ติดตั้งเป็นแอปบนมือถือ (PWA)
- เปิดเว็บแอปผ่าน URL ที่ deploy จริง (เช่น GitHub Pages, Netlify, Vercel — ต้องเป็น HTTPS)
- Android/Chrome: เมนู ⋮ → "ติดตั้งแอป" หรือ "Add to Home screen"
- iPhone/Safari: ปุ่มแชร์ → "เพิ่มไปยังหน้าจอโฮม"

### Deploy ฟรีแบบง่าย ๆ
วิธีที่ง่ายที่สุดคือ push โฟลเดอร์นี้ขึ้น GitHub แล้วเปิดใช้ **GitHub Pages** (Settings → Pages → เลือก branch) หรือลากโฟลเดอร์ไปวางที่ **Netlify Drop** (https://app.netlify.com/drop)

## 🤖 ตั้งค่าฟีเจอร์ AI (สรุป/เรียบเรียงข้อความ)

แอปนี้รองรับทั้ง **Anthropic (Claude)** และ **OpenAI**

1. ไปที่แท็บ **"ตั้งค่า"** ในแอป
2. เลือกผู้ให้บริการ แล้ววาง API Key ของคุณ:
   - Anthropic: สมัคร/สร้าง key ได้ที่ https://console.anthropic.com/
   - OpenAI: สร้าง key ได้ที่ https://platform.openai.com/api-keys
3. กด "บันทึกการตั้งค่า AI"
4. กลับไปหน้าบันทึกประจำวัน แล้วกดปุ่ม "ให้ AI ช่วยเรียบเรียงเป็นภาษาราชการ"

**⚠️ ข้อควรระวังด้านความปลอดภัย:** แอปนี้เรียก API โดยตรงจากเบราว์เซอร์ (client-side) เพื่อความง่ายในการใช้งานส่วนตัว ซึ่งหมายความว่า API Key จะถูกเก็บไว้ใน localStorage ของเครื่องคุณเท่านั้น (ไม่ส่งไปที่ไหนอื่น) แต่ก็ไม่ควรแชร์ไฟล์/ลิงก์แอปที่ deploy แล้วให้คนอื่นใช้ร่วมกับ key ของคุณ ถ้าจะใช้งานแบบสาธารณะจริงจัง ควรทำ backend เล็ก ๆ (เช่น Cloudflare Worker หรือ Firebase Function) มาซ่อน API key แทนการเรียกตรงจาก client

## ☁️ ตั้งค่าซิงก์กับ Firebase (ไม่บังคับ)

เพื่อให้ข้อมูลไม่หายเมื่อเปลี่ยนเครื่อง:

1. ไปที่ https://console.firebase.google.com/ → สร้างโปรเจกต์ใหม่ (ฟรี)
2. ในโปรเจกต์ → **Build → Firestore Database** → Create database (เลือก production/test mode ก็ได้)
3. ไปที่ **Build → Authentication → Sign-in method** → เปิดใช้ **Anonymous**
4. ไปที่ Project settings (รูปเฟือง) → ส่วน "Your apps" → เพิ่ม Web App (</>) → คัดลอกค่า `firebaseConfig` ที่ได้ (หน้าตาแบบนี้):
   ```json
   {
     "apiKey": "...",
     "authDomain": "...",
     "projectId": "...",
     "storageBucket": "...",
     "messagingSenderId": "...",
     "appId": "..."
   }
   ```
5. นำ JSON นี้ไปวางในแอป → แท็บ "ตั้งค่า" → ช่อง "ซิงก์ข้อมูลกับ Firebase" → กด "บันทึกการตั้งค่า"
6. กด **"ซิงก์ตอนนี้"** เพื่ออัปโหลดข้อมูล หรือ **"ดึงข้อมูลจากคลาวด์"** เมื่อเปิดจากเครื่องใหม่

> หมายเหตุ: ระบบใช้ Firebase Anonymous Auth เพื่อความง่าย ซึ่งหมายความว่าแต่ละเบราว์เซอร์/อุปกรณ์จะได้บัญชีชั่วคราวของตัวเอง หากต้องการให้ข้อมูล "ตามตัวคุณ" ข้ามอุปกรณ์แบบเป๊ะ ๆ แนะนำให้ปรับ `js/firebase-sync.js` ให้ใช้ `signInWithEmailAndPassword` หรือ Google Sign-In แทน (เพิ่มปุ่ม login แล้วเรียก `firebase.auth().signInWithPopup(...)`)

```
internship-app/

│   ├── icon-192.png

    ├── db.js                 จัดการข้อมูลทั้งหมดผ่าน localStorage
    ├── voice.js               อัดเสียง → ข้อความ (Web Speech API)
    ├── ai.js                   เรียก Anthropic/OpenAI API

```

## 🛠️ ปรับแต่งเพิ่มเติมที่แนะนำ

- **เปลี่ยนชื่อบริษัท/ชื่อผู้ฝึกงานในรายงาน**: แก้ไขใน `js/app.js` ฟังก์ชัน `exportDocx`/`exportPdfViaPrint` (ส่ง `meta.name`, `meta.org` เพิ่มได้ — ปัจจุบันยังไม่มีช่องกรอกใน UI ลองเพิ่มในแท็บตั้งค่าได้)
- **แปลง PDF โดยไม่ผ่านหน้าต่างพิมพ์**: ถ้าต้องการดาวน์โหลดไฟล์ PDF ตรง ๆ โดยไม่เปิด dialog พิมพ์ ให้พิจารณาใช้ไลบรารีอย่าง `pdfmake` พร้อมฝังฟอนต์ไทย (Sarabun) เอง เพราะ jsPDF ไม่รองรับฟอนต์ไทยในตัว
- **แปลงเสียงด้วย AI จริง (Whisper)**: ปัจจุบันใช้ Web Speech API ของเบราว์เซอร์ซึ่งฟรีและไม่ต้องมี key แต่ความแม่นยำอาจน้อยกว่า Whisper — ถ้าต้องการ ให้เพิ่มฟังก์ชันอัดเสียงเป็นไฟล์ (MediaRecorder) แล้วส่งไปที่ OpenAI Whisper API แทน
